야구경기(KBO) 결과 예측

상세 내용 : Link

개요

2015~2019 KBO리그 데이터를 이용해 2020년 KBO매치 결과 예측

상세 내용

임의의 두 팀의 경기에서 승패를 예측하기 위해서는 양팀간의 상대전적, 최근 기세 등을 고려해야할 것이다.
여기서는 최근 50경기동안의 각 팀의 평균타율, 득점, 실점 등의 가중평균(최근 경기에 가중치를 둠)을 최근 기세 지표로 사용했다. KBO리그는 각 팀이 1년간 144경기를 진행하고, 총 10개팀이 존재하기 때문에 매년 KBO는 720경기가 진행된다. 최근 50경기의 가중평균값을 지표로사용하기때문에 각 팀당 94개 경기의 데이터를 실질적으로 사용할 수 있었다.
분석 모델은 로지스틱회귀, 의사결정나무, 랜덤포레스트, 딥러닝 총 4개를 사용했으며 평가지표는 정확도(Accuracy)를 사용했다. 결과는 테스트셋(2020년)에서 정확도 55%로 다소 아쉬운 결과를 내었다.
